# 2D Wizards vs Goblins

A 2D action game where players take control of a wizard to battle waves of goblins. Players can cast spells, dodge attacks, and survive as long as possible. Currently, all core gameplay mechanics are functional, though upgrading abilities and stats is a planned future feature.

## Gameplay

In **2D Wizards vs Goblins**, you play as a wizard defending your territory from incoming goblins. The game combines strategy, timing, and skill as you cast spells and avoid enemy attacks. Survive as long as possible while defeating waves of goblins to achieve a high score.

---

## Features

- Fully functional player movement and controls.  
- Multiple goblin enemies with basic AI.  
- Attack system with spells.  
- Health and damage system for both player and enemies.  
- Score tracking for waves defeated.  

**Note:** Upgrading abilities and player stats is not yet implemented but is planned for future updates.

---

## Controls

| Action        | Key / Input      |
|---------------|----------------|
| Move Up       | W / Up Arrow   |
| Move Down     | S / Down Arrow |
| Move Left     | A / Left Arrow |
| Move Right    | D / Right Arrow|
| Cast Spell    | Left Click|
